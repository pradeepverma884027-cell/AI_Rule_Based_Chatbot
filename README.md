# 🤖 AI Rule-Based Chatbot

A simple Rule-Based AI Chatbot developed using Python as part of the **DecodeLabs Artificial Intelligence Internship - Project 1**.

The chatbot interacts with users using predefined rules and responses. It demonstrates the fundamentals of Artificial Intelligence through decision-making using Python's `if-elif-else` statements.

---

## 📌 Project Objective

The objective of this project is to build a basic AI chatbot that:

- Responds to predefined user inputs
- Uses decision-making (`if-elif-else`)
- Runs continuously until the user exits
- Demonstrates the basic concepts of Artificial Intelligence

---

## ✨ Features

- Interactive command-line chatbot
- Continuous conversation using `while True`
- Handles greetings
- Answers predefined questions
- Accepts multiple exit commands
- Handles unknown inputs gracefully
- User-friendly interface
- Simple and easy to understand
